# How to Run

## Clone Repository

```bash
git clone https://github.com/<your-username>/CTS_Deepskilling.git
```

## Java Projects

```bash
cd Week1
mvn clean install
```

or

```bash
mvn spring-boot:run
```

## Angular Project

```bash
cd Week6/student-course-portal

npm install

ng serve
```

Run Unit Tests

```bash
ng test
```
